# README

This project mainly provides compilation packages of some commonly used linux software on Ubuntu.

Compiled software:

- nginx

- redis

- mysql

- pgsql

- mqtt(mosquitto)

- strongswan(IKEv2 VPN)

- openvpn(OpenVPN)

Compiled tools:

- dlv

General tools:

- file proxy download

- vedio download(m38u)
